  Dependencies :<br>
 	1. sleep<br>
 	2. curl<br>
 	3. shc<br>
 	4. zenity<br>
 
  Install shc :<br>
 	1. su - root<br>
 	2. wget http://www.datsi.fi.upm.es/~frosal/sources/shc-3.8.7.tgz<br>
 	3. tar xvfz shc-3.8.7.tgz<br>
 	4. cd shc-3.8.7<br>
 	5. make<br>
 	6. cp shc /usr/bin/<br>
 
  Usage :<br>
 	1. chmod +x install.sh<br>
 	2. ./install.sh<br>
 	3. when Gcheck.sh has created, start up this file in your system<br>
